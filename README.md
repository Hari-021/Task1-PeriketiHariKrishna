# Nexus University Smart Campus Digital Twin

A futuristic 3D Smart Campus Digital Twin website built for modern universities. This project combines an interactive Three.js campus visualization with premium glassmorphism UI, real-time dashboards, smart navigation, analytics, events, and an AI-style campus assistant.

## Features

- Interactive 3D campus visualization using Three.js and React Three Fiber
- Clickable campus buildings with live information panels
- Animated route paths connecting departments and facilities
- Day and night mode toggle
- Smart classroom, lab, library, canteen, and office search
- Route finder and building directory
- Student dashboard with attendance, timetable, assignments, and performance analytics
- Faculty dashboard with attendance management and department statistics
- Analytics dashboard with occupancy charts, activity metrics, heatmaps, and insights
- Events module with registrations, countdowns, gallery cards, and analytics
- Floating AI campus assistant UI
- Responsive design for mobile, tablet, and desktop
- Dark glassmorphism theme with neon blue, purple, and accent green styling
- Smooth page transitions and micro-interactions using Framer Motion

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- Three.js
- React Three Fiber
- React Three Drei
- Framer Motion
- React Router
- Lucide React Icons
